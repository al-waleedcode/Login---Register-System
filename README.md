# 🔐 Frontend Authentication System 

A complete, single-page Login and Registration interface built with **Vanilla JavaScript** and **Tailwind CSS**. This project demonstrates how to handle user sessions and data validation on the client side without relying on a backend server, utilizing browser `LocalStorage`.

## 🛡️ Security & Validation Features
* **Input Validation:** Enforces strict password length and email formatting before form submission to prevent bad data entry.
* **Duplicate Prevention:** Checks existing local data to ensure no two accounts can register with the same email address.
* **Secure Flow Mockup:** Simulates a secure authentication flow, dynamically toggling between Login and Registration views without page reloads.

## 🚀 Technical Highlights
* **State Management:** Uses JavaScript to seamlessly switch UI states and display contextual success/error alerts.
* **Data Persistence:** Effectively uses `localStorage` to save user credentials (mocking a database) and verifies them during login.
* **Responsive UI:** Modern, clean, and mobile-friendly design achieved through utility-first Tailwind CSS.

## 🛠️ Tech Stack
* **HTML5** - Semantic structure.
* **Tailwind CSS** - Rapid UI styling.
* **JavaScript (ES6+)** - Core logic, DOM manipulation, and validation.
